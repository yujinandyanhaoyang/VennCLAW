# 论文引用格式说明

**标准**：GB/T 7714-2015 信息与文献 参考文献著录规则

---

## 期刊论文格式

```
[序号] 作者。论文标题 [J]. 期刊名，年，卷 (期): 起止页码.
```

**示例**：
```
[4] Vaswani A, Shazeer N, Parmar N, et al. Attention Is All You Need[J]. 
    Advances in Neural Information Processing Systems, 2017, 30.
```

---

## 专著/图书格式

```
[序号] 作者。书名 [M]. 版本。出版地：出版社，出版年：起止页码.
```

**示例**：
```
[1] Wooldridge M. An Introduction to MultiAgent Systems[M]. 2nd ed. 
    John Wiley & Sons, 2009.
```

---

## 会议论文格式

```
[序号] 作者。论文标题 [C]//会议论文集名。出版地：出版社，出版年：起止页码.
```

**示例**：
```
[6] Li Y, Zhang H, Wang X. Multi-Agent Collaboration for Complex Task 
    Solving[C]//International Conference on Learning Representations (ICLR), 2023.
```

---

## 电子资源格式

```
[序号] 作者。标题 [EB/OL]. (更新日期)[引用日期]. URL.
```

**示例**：
```
[10] Tiangolo S. FastAPI Documentation[EB/OL]. (2023)[2026-03-11]. 
     https://fastapi.tiangolo.com/
```

---

## arXiv 预印本格式

```
[序号] 作者。标题 [R]. arXiv preprint arXiv:编号，年份.
```

**示例**：
```
[8] McCallum A. Tool-Augmented Language Models[R]. 
    arXiv preprint arXiv:2209.12345, 2022.
```

---

## 注意事项

1. **作者姓名**：英文作者姓在前、名在后，名可缩写（如 Wooldridge M）
2. **三人以上**：列出前三位作者，后加", et al."
3. **标点符号**：使用英文标点，各字段间用"."分隔
4. **斜体使用**：期刊名、书名使用斜体（Markdown 中用*包裹）
